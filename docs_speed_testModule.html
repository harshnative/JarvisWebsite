Speed Test

Jarvis Includes a inbuilt in utility to test your internet speed.

Command's

  1. speed test

    To start speed test with default settings

  2. speed test -b

    To get speed test result in mega bytes instead of mega bits

  3. speed test 3

    To average the speed test result 3 times instead of defualt (2) times

    Note : Value above 5 is ignored

  4. speed test -b 3

    You can combine the commands into one
