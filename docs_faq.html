Weather

  1. I want to get weather detail of new york and after using this command i get error message ?.

    weather city-new york

    Please use underscore instead of city name to write city name having two or more words. Use weather city-new_york.

BackUp

  1. Jarvis is not backing up some of my files ?.

    Unsupported files like encrypted container and files that are not accessible due to OS permission issue are not backed up.

  2. Jarvis is not backing up my directories mentioned in the settings file ?.

    Make sure that the paths are correct and you have read permission for that folder.

    Path examples - C:/Users/userName/Desktop/folderName.

    If you have mulitple directories then make sure to separate them out using colon ( , ).

  3. How does Jarvis backup works ?.

    Jarvis copy all the files in the directory and all the subdirectories to the backUp path , if the files are already in the backup then the files will be replaced by the new files.

  4. Jarvis backup not working ?.

    Make sure that the path to backup in the settings is set and is correct.

    Path examples - D:/myFiles.

    A folder will be created in the myFiles called backup in which your backed up data will be present.

  5. Where is my directory backup in the backup folder ?.

    Your directories backup will be located under the additional files folder in the backup folder.

  6. Jarvis seems to be stuck while Backing Up Data ?.

    Copying files takes time and depends on the computer processing power and transfer speed btw drives, jarvis may be still running, you can conform that by opening task manager.

Password Manager

  1. Password input is not accepting inputs ?.

    Jarvis password input is just like Linux , it is taking input and recording it but not showing it visually. This is a security feature to prevent password leaks.

  2. I am not able to delete or update anything ?.

    Make sure you enter correct index number when asked about them
